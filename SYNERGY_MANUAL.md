## 🚀 COMMAND REFERENCE GUIDE

### 🔥 Ignition & Shutdown
- **Full Start:** `./synergy-startup.sh` (Clears ports, syncs frontend, starts backend)
- **Hard Stop:** `pkill -f synergy_ui` (Kills all backend processes)
- **Diagnostic:** `./synergy-doctor.sh` (Checks all dependencies)

### 📦 Database Management
- **Migrate DB:** `cd backend && venv/bin/alembic upgrade head`
- **Reset Redis:** `redis-cli flushall`

### 🔧 Ollama Controls
- **List Models:** `ollama list`
- **Pull Model:** `ollama pull [model_name]`
- **Show Logs:** `journalctl -u ollama --no-pager -n 50`

## 🔐 WINDOWS SECURITY HOTFIX
If you see "Blocked App" or "ggml-cuda.dll" errors:
1. Open **Windows Security**.
2. **Virus & Threat Protection** > **Manage Settings** > **Exclusions**.
3. Add Folder: `C:\Users\DevOps\AppData\Local\Programs\Ollama`
4. Add Process: `ollama.exe`
5. Restart Ollama from the Windows Task Tray (right-click exit, then relaunch).
